# Shortcuts

### Prefix Key for Shortcuts

Cntrl+B

### General Purpose

Cntrl + B ? : Show all shortcuts available

### Window Shortcuts

Cntrl + B c : Create a new window
Cntrl + B n : Go to next window
Cntrl + B & : Kill Window

### Pane Shortcuts

Cntrl + B % : Vertical Split
Cntrl + B " : horizontal split
